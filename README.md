# Django Tweet Project

A simple Django-based web application that allows users to create, edit, 
and delete tweets. Each tweet includes a name, description, and an image, 
making it a lightweight CRUD (Create, Read, Update, Delete) project built 
to demonstrate core Django concepts.

## Features
- Create new tweets with a name, description, and image
- Edit existing tweets
- Delete tweets
- Image upload and display support
- Built using Django's ORM and admin panel

## Tech Stack
- Python
- Django
- SQLite (development database)
- HTML/CSS (templates)

## Getting Started

1. Clone the repository
```bash
   git clone https://github.com/your-username/django-tweet-project.git
```

2. Create and activate a virtual environment
```bash
   python -m venv .venv
   .venv\Scripts\activate   # Windows
```

3. Install dependencies
```bash
   pip install -r requirements.txt
```
